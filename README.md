# Landing Page - Task 1

## Objective
To create a beginner-friendly, responsive landing page clone to demonstrate a foundational understanding of web development structure and styling. 

## Steps Performed
1. **Structuring:** Built the foundational skeleton of the webpage using semantic HTML5 tags (`<header>`, `<section>`, `<footer>`) to ensure readability.
2. **Styling & Layout:** Applied CSS3 to style the elements. Utilized CSS Flexbox extensively to create a responsive, non-overlapping layout for the navigation, feature rows, and benefits grid.
3. **Refining:** Added hover states to buttons and adjusted padding/margins to match the provided design aesthetic.
4. **Deployment:** Pushed the code to GitHub and deployed the live site using GitHub Pages.

## Tools Used
*   HTML5
*   CSS3
*   Git & GitHub (for version control and hosting)
*   VS Code

## Outcome in Brief
Successfully developed a clean, modern, and visually balanced landing page without relying on JavaScript or external CSS frameworks. The project solidifies core competencies in CSS Flexbox and responsive design principles.
